# Data Science in Python: Data Preparation & EDA

This repository documents my learning journey through a project-based course focused on the core building blocks of data science using Python. It covers the complete workflow from data gathering to exploratory data analysis (EDA) and feature engineering for machine learning.

## 📌 Overview

The course introduces fundamental concepts in data science and machine learning, followed by a deep dive into data preparation and exploratory analysis. All work in this repository is based on practical exercises and projects designed to simulate real-world data science tasks.

Throughout the course, I work as a Jr. Data Scientist for a fictional company, **Maven Music**, analyzing customer data to uncover insights related to customer churn.

## 🧠 Key Learning Areas

- Understanding the Data Science Workflow
- Supervised vs Unsupervised Learning
- Project Scoping Techniques
- Data Collection from Multiple Sources
- Data Cleaning and Preprocessing using Pandas
- Exploratory Data Analysis (EDA)
  - Filtering and grouping data
  - Aggregations
  - Data visualization
- Data Preparation for Machine Learning
  - Joining datasets
  - Adjusting data granularity
  - Feature engineering

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Jupyter Notebook
- Data Visualization Libraries (Matplotlib / Seaborn)
